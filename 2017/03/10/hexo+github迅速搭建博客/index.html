hexo+github迅速搭建博客

  • 时间：2017-03-10
    • 博客

导语

随着所学的编程技术越来越多,最近萌生出写博客的念头,以整理这几年所学,同时达到温故而知新的目的.

起初考虑过用WordPress搭建一个博客,或者使用CSDN,博客园.但是用WordPress首先要有主机,需要管理的东西也太多,各种各样的配置,感到十分麻烦.而后者又门槛太低,不够优雅不够geek.

于是想到用hexo,部署到GitHub page上,再绑定域名即可.省钱省力,自由而优雅.

准备

我所用的是elementary os系统,一个基于Ubuntu的Linux发行版.因此安装hexo环境十分便捷

首先更新软件

$ sudo apt-get update

安装git

命令行执行

$ sudo apt-get install git

安装nodejs

由于hexo由nodejs编写,需要安装nodejs.

$ sudo apt-get install

安装hexo

$ npm install hexo

建站

一次执行下列命令即可,十分便捷.

$ hexo init 文件夹
$ cd 文件夹
$ npm install

本地测试

执行完后便建好了本地hexo文件,通过以下命令,生成静态页面.

$ hexo g

启动本地服务器

$ hexo server

默认访问: http://localhost:4000/ ,便可本地测试网站.

部署到github

首先必须先注册github账号.

然后创建好代码库.

进入hexo网站安装文件夹根目录执行:

$ sudo vi _config.yml

按照一下配置:

deploy:
type: git
repo: github仓库访问地址

注意!!!

缩进
YAML依靠缩进来确定元素间的从属关系。因此，请确保每个deployer的缩进长度相同，并且使用空格缩进。

最后,执行下条命令即部署到github:

$ hexo d

结语

在架设博客中,肯定会碰见各种各样的问题,如果仅仅按照上述步骤,在不是很了解hexo的情况下,100%会出现错误以上只是简单的叙述hexo+github搭建博客.具体仍是建议参考官网文档.
