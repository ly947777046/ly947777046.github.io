终于还是入了Linux的坑

  • 时间：2017-03-16
    • linux

大概在两年前，第一次接触到Linux发行版Center OS。给我的第一印象是，卧槽，系统竟然还有免费的！再稍微了解，卧槽，感觉好高端！我要装上！

于是，赶紧做了个u盘启动，磕磕绊绊的真装上了。当空荡荡的桌面显示出来后，一脸懵逼，这鬼东西怎么使？而且那时还不知道有mentohust之类东西，完全拿校园网没办法。然后无奈的又装回Windows。可惜了我满硬盘的珍藏小电影啊。

与Linux的初次会面就这样不愉快地say goodbye。

后来，一年前偶然发现了深度Linux系统，于是又怀着不作不得劲的精神又折腾上了。当时尤其好奇系统还能像装软件一样给装上！没多久，又发现了个叫Ubuntu的系统。装上！不过学聪明了，搞双系统。后来嘛……因为双系统引导问题又是一番折腾。再后来，又发现deepin比Ubuntu漂亮多了……

由始至终，那段时间一直是把Linux当玩具玩。虽然确确实实感觉到了是比Windows流畅多了。然并卵，我还是只会在Windows敲代码。并且总会抱怨为啥Linux没有QQ没有迅雷没有……

慢慢的，用到的工具越来越多，装的软件也越来越多，我那装着盗版Wind10的笔记本越来越不堪重负了。最重要的是，也越来越忍受不了在找资料的时候老是会碰上流氓程序给我装些乱七八糟的东西。更是不能忍Windows上管家之类的程序老蹦提示又不得不装上保护系统安全。于是弃了。

找了个据说比Ubuntu好看又更流畅的Ubuntu mate。确实更好看丝滑流畅！就是汉字是断脚的越看越难受。忍了，配置好JAVA开发环境，认真敲代码。没过几天，实在忍不了，加上搭建hexo总是出莫名其妙的问题换上了公认最美桌面的elementary OS。同是debain系Linux，不用改变习惯用的倒也顺手。好好地把玩了几天这个看起来像是Mac的系统后，该搞正事了，好好敲代码才是真理。当装好一切，想要打开eclipse，what the fuck？卡死！查了资料说是桌面系统的问题，改eclipse.ini强制gtk2启动吧，显示得一副鬼样子。

深刻理解，Linux上折腾桌面就是闲的蛋疼。换回了elementary OS的爸爸Ubuntu，装啥都顺利了。尽管也会出些问题，但Google一下手把手教程一片一片的，折腾了那么久，更懂得了沉心静气去看log才是办法！

用Linux也是有段日子了，越发感受到Linux的魅力。从最初的把玩桌面，到越过图形界面敲命令行，才算是开始了解到Linux真正使用Linux。命令用的多了，才发现，图形能做的事，命令全都能做到，反之，则不行。Linux的程序不是都有图形界面的，更多的，藏在命令里，小而强大，可单独用，可以组合起来用。甚至编辑器都实在命令行，著名如vim。太多太多，使用Linux永远在学习。

最后，我想说，用Linux不是为了装逼，是为了效率……归根结底，还是因为没钱买不起Mac。
